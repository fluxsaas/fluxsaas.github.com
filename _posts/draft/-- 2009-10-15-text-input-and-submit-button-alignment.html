How do i align an submit button and a text field in a form ?

{% highlight xml %}
{% endhighlight %} {% highlight css %} form input{ font-size: 11px; margin: 0 0 0 2px; width: 150px; height: 13px; } form button{ vertical-align: bottom; } *html form button { position: relative; top: -1px; } {% endhighlight %}

The small tips are the Best: Text input and submit button alignment. THX !
